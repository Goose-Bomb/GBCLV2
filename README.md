# GBCLV2
* A Simple Win10-Style Minecraft Launcher<br>   
* Based on C# WPF and .NET Framework V4.7.1<br>  
* Using [Modified KMCCC](https://github.com/Goose-Bomb/KMCCC) as launcher core<br>
